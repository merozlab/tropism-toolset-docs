## Requirements

- **Python**: 3.13 (required)
- **Package Manager**: uv (recommended) or pip

## Installation Methods

<Tabs>
<TabItem label="uv (Recommended)">

[uv](https://docs.astral.sh/uv/) is a fast Python package installer and resolver that provides better dependency management and faster installation times.

```bash
# Install uv if you haven't already
curl -LsSf https://astral.sh/uv/install.sh | sh

# Navigate to the project directory
cd path/to/tropism-toolset

# Create a virtual environment and install the package
uv sync

# Activate the virtual environment
source .venv/bin/activate
```

</TabItem>
<TabItem label="pip">

If you prefer using pip:

```bash
# Navigate to the project directory
cd path/to/tropism-toolset

# Create a virtual environment
python -m venv .venv

# Activate the environment
source .venv/bin/activate  # On macOS/Linux
# or
.venv\Scripts\activate     # On Windows

# Install the package in editable mode
pip install -e .
```

</TabItem>
</Tabs>

## Jupyter Setup

To use the toolkit in Jupyter notebooks, you need to register the environment as a Jupyter kernel:

```bash
# With the virtual environment activated
python -m ipykernel install --user --name tropism-toolset-env --display-name "Python (Tropism Toolset)"
```

After this, you can select "Python (Tropism Toolset)" as your kernel when running Jupyter notebooks.

## Dependencies

The toolkit includes the following dependencies:

- **NumPy** (≥2.2.5): Numerical computing
- **Matplotlib** (≥3.10.3): Plotting and visualization
- **SciPy** (≥1.15.3): Scientific algorithms and optimization
- **pandas** (≥2.3.3): Data manipulation
- **scikit-learn** (≥1.7.0): Data analysis and fitting
- **OpenCV** (≥4.11.0): Image processing
- **Altair** (≥5.5.0): Interactive visualizations
- **PySide6** (≥6.9.0): GUI components
- **NetworkX** (≥3.5): Graph algorithms
- **openpyxl** (≥3.1.5): Excel file handling
- **ipykernel** (≥7.1.0): Jupyter notebook support

## Verifying Installation

To verify that the installation was successful, try importing the package:

```python
from tropism_toolset import (
    get_angles,
    get_arclengths,
    fit_Lc,
    get_gamma,
    get_beta,
    plot_centerline_data
)

print("Installation successful!")
```
