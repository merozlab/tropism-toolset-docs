The Moulia model combines proprioceptive (self-sensing of curvature) and gravitropic responses in a unified framework.

## Overview

This model describes how plants integrate:
- **Gravitropic sensing**: Response to gravity direction
- **Proprioceptive sensing**: Awareness of their own curvature

## Implementation

The primary analysis notebook for this model is `Moulia_constants.ipynb` in the repository.

### Key Equations

The model relates curvature changes to both gravitropic and proprioceptive terms:

<Equation>

$$\frac{\partial \kappa}{\partial t} = \beta g \sin\theta - \gamma \kappa$$

</Equation>

Where:
- $\kappa$ = curvature
- $\beta$ = gravitropic sensitivity
- $\gamma$ = proprioceptive sensitivity
- $\theta$ = angle from vertical
- $g$ = gravitational acceleration
