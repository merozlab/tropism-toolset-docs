The Chauvet model describes how plants sense **inclination** rather than force during gravitropic responses.

## Theory

### Key Finding

> **Plants sense inclination, not force**

Chauvet et al. (2016) demonstrated that the gravitropic response depends on the organ's angle relative to vertical, not the gravitational force itself.

**Reference:**
Chauvet, H., Pouliquen, O., Forterre, Y. et al. *Inclination not force is sensed by plants during shoot gravitropism.* Sci Rep 6, 35431 (2016).
https://doi.org/10.1038/srep35431

### Mathematical Framework

The relative differential growth is defined as:

<Equation>

$$\tilde{\Delta} = \frac{R}{\dot{E}} \frac{\partial \kappa}{\partial t}$$

</Equation>

Where:
- $R$ = radius of the organ
- $\dot{E}$ = local growth rate (s⁻¹)
- $\kappa(s,t)$ = local curvature (m⁻¹)

### Homogenized Growth Rate

Following Bastien et al., the mean relative growth rate can be homogenized over the growth zone:

<Equation>

$$\dot{E} \approx \frac{1}{L_{gz}} \frac{dL}{dt}$$

</Equation>

Where:
- $L_{gz}$ = length of the growth zone
- $L$ = total organ length

### Curvature Approximation

For initially straight organs:

<Equation>

$$\kappa \approx \frac{\theta_{tip} - \theta_0}{L_{gz}}$$

</Equation>

Where:
- $\theta_{tip}$ = tip angle relative to vertical
- $\theta_0$ = base angle

### Simplified Form

For classic gravitropism experiments (initially horizontal, $\theta_0 = \pi/2$):

<Equation>

$$\tilde{\Delta} = R \frac{d\theta_{tip}}{dt} / \frac{dL}{dt}$$

</Equation>

### Sine Law

The gravitropic response follows:

<Equation>

$$\tilde{\Delta} = \tilde{\beta} \sin\theta_0$$

</Equation>

Where $\tilde{\beta}$ is the gravitropic sensitivity.

For horizontal organs ($\sin\theta_0 = 1$):

<Equation>

$$\tilde{\beta} = R \frac{d\theta_{tip}}{dt} / \frac{dL}{dt}$$

</Equation>

## Implementation

### Step 1: Extract Growth Rate

```python
from constants import fit_growth_rate, get_arclengths
import pandas as pd
import numpy as np

# Load data
data = pd.read_csv("centerlines.csv")
px_to_m = 100

# Extract lengths over time
frames = []
lengths = []

for frame in sorted(data['frame'].unique()):
    frame_data = data[data['frame'] == frame]
    arclengths = get_arclengths(frame_data) / px_to_m  # Convert to meters
    frames.append(frame)
    lengths.append(arclengths[-1])

frames = np.array(frames)
lengths = np.array(lengths)

# Fit growth rate (dL/dt)
growth_rate = fit_growth_rate(
    frames,
    lengths,
    show=True,
    ylabel="Length (m)",
    title="Growth Rate",
    slope_units="m/frame"
)

print(f"dL/dt = {growth_rate:.6f} m/frame")
```

### Step 2: Extract Angular Velocity

```python
from constants import fit_angular_velocity, get_angles

# Extract tip angles over time
tip_angles = []
for frame in sorted(data['frame'].unique()):
    frame_data = data[data['frame'] == frame]
    angles = get_angles(frame_data)
    tip_angles.append(angles[-1])

tip_angles = np.array(tip_angles)

# Fit angular velocity (dθ/dt)
angular_velocity = fit_angular_velocity(
    frames,
    tip_angles,
    show=True,
    title="Angular Velocity"
)

print(f"dθ/dt = {angular_velocity:.6f} rad/frame")
```

### Step 3: Calculate β̃

```python
# Plant radius (typical root: 1-2 mm)
R = 0.00145  # 1.45 mm in meters

# Calculate gravitropic sensitivity
beta_tilde = R * angular_velocity / growth_rate

print(f"\nChauvet Model Results:")
print(f"Radius (R): {R*1000:.2f} mm")
print(f"Growth rate (dL/dt): {growth_rate:.6f} m/frame")
print(f"Angular velocity (dθ/dt): {angular_velocity:.6f} rad/frame")
print(f"Gravitropic sensitivity (β̃): {beta_tilde:.4f} m⁻¹")
```

### Step 4: Convert to Biological Units

```python
minutes_per_frame = 15
seconds_per_frame = minutes_per_frame * 60

# Convert rates to per-second
dL_dt_per_sec = growth_rate / seconds_per_frame
dtheta_dt_per_sec = angular_velocity / seconds_per_frame

print(f"\nBiological Units:")
print(f"Growth rate: {dL_dt_per_sec*1e6:.2f} μm/s")
print(f"Angular velocity: {np.degrees(dtheta_dt_per_sec):.4f} °/s")
print(f"Angular velocity: {np.degrees(dtheta_dt_per_sec)*3600:.2f} °/hour")
```

## Physical Interpretation

### β̃ (Beta Tilde)

The gravitropic sensitivity represents how strongly the plant responds to gravitational cues:

- **High β̃**: Strong gravitropic response, rapid reorientation
- **Low β̃**: Weak response, slow reorientation
- **Typical values**: 0.1 - 10 m⁻¹ for roots

### Relationship to Growth

The ratio $\frac{d\theta/dt}{dL/dt}$ tells us:
- How much bending occurs per unit of growth
- The balance between elongation and reorientation
- The efficiency of the gravitropic mechanism

## Experimental Considerations

### Initial Orientation

The model assumes:
- Initially straight organ
- Known initial inclination $\theta_0$
- For horizontal stimulation: $\theta_0 = \pi/2$

### Growth Zone

The model works best when:
- Growth is confined to apical zone
- Growth rate is approximately constant
- Curvature changes occur primarily in growth zone

### Measurement Requirements

For accurate β̃ calculation:
1. High temporal resolution (capture $d\theta/dt$ accurately)
2. Precise length measurements (reliable $dL/dt$)
3. Accurate radius measurement
4. Consistent experimental conditions

## Complete Analysis Script

```python
from constants import *
import pandas as pd
import numpy as np

# Configuration
px_to_m = 100
minutes_per_frame = 15
R = 0.00145  # meters

# Load data
data = pd.read_csv("experiment.csv")

# Extract time series
frames, lengths, tip_angles = [], [], []
for frame in sorted(data['frame'].unique()):
    fd = data[data['frame'] == frame]
    arclengths = get_arclengths(fd) / px_to_m  # Convert to meters
    angles = get_angles(fd)

    frames.append(frame)
    lengths.append(arclengths[-1])
    tip_angles.append(angles[-1])

frames = np.array(frames)
lengths = np.array(lengths)
tip_angles = np.array(tip_angles)

# Fit rates
growth_rate = fit_growth_rate(frames, lengths, show=True)
angular_velocity = fit_angular_velocity(frames, tip_angles, show=True)

# Calculate Chauvet parameter
beta_tilde = R * angular_velocity / growth_rate

# Report
print("\n" + "="*60)
print("CHAUVET MODEL ANALYSIS")
print("="*60)
print(f"Radius (R):                    {R*1000:.2f} mm")
print(f"Growth rate (dL/dt):           {growth_rate:.6e} m/frame")
print(f"Angular velocity (dθ/dt):      {angular_velocity:.6e} rad/frame")
print(f"Gravitropic sensitivity (β̃):  {beta_tilde:.4f} m⁻¹")
print("="*60)
```

## Advantages

1. **Simple**: Only requires tip angle and length tracking
2. **Direct**: No complex curve fitting
3. **Physical**: Clear interpretation in terms of growth and bending
4. **Validated**: Experimentally verified across species

## Limitations

1. Assumes initially straight organ
2. Requires constant growth rate
3. Sensitive to measurement noise
4. Growth zone must be localized
